Part 3. IMPLEMENTATION

The dataset of this webapp includes 26 separate CSV files and one dataset array in the javascript file. All these datasets were created by R based on the data subtraction and processing result of Nextstrain and WHO Coronavirus databases. Generally speaking, I selected specific attributes from the original dataset and then accumulate the required attributes by month. Besides, the monthly deaths were divided by the monthly cases to partially represent the Case Fatality Rate.

Adapted from Martin Chorley's Block at https://bl.ocks.org/martinjc/8a2a545d5c48ef1ff65b747554332e3d.

These implemented visualizations allow users to use a slider or dropdown list to interact with the visualization result. The interaction triggers transitions and the changing of datasets.

Acknowledgement of Data Source

The dataset for newly increased COVID-19 cases and deaths is from the WHO Coronavirus (COVID-19) Dashboard at https://covid19.who.int/data.
